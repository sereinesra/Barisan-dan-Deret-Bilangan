    • POLA BILANGAN SEGITIGA


    Selain membentuk pola persegi dan persegi panjang, pola bilangan juga memmbentuk pola segitiga. Perhatikanlah gambar di bawah untuk memahami pola segitiga.

    Gambar 2. 10 Pola Bilangan Segitiga


    Setelah melihat gambar pola segitiga di samping, dapatkah kamu merumuskan bagaimana pola yang terbentuk?


    Perhatikanlah tabel di bawah untuk memahaminya lebih lanjut.

    Tabel 2. 4

    洧녣洧녵 Nilai
    洧녣1 1 = (1 + 1) x 1/2 = 2/2
    洧녣2 3 = (2 + 1) x 2/2 = 6/2
    洧녣3 6 = (3 + 1) x 3/2 = 12/2
    洧녣4 10 = (4 + 1) x 4/2 = 20/2
    ... ...
    洧녣洧녵 (洧녵 + 1) x 洧녵/2

    Berikut rumus pola bilangan segitiga:

    (洧녵 + 1) x 洧녵/2


    CONTOH SOAL


    1. Tentukan suku ke-12, suku ke-20, dan suku ke-35 pada pola bilangan segitiga!

      2. Jawab:

      • 洧녣12 = (12 + 1) 洧논 12/2

        洧녣12 = 13 洧논 12/2

        洧녣12 = 78

      • 洧녣20 = (20 + 1) 洧논 20/2

        洧녣20 = 21 洧논 20/2

        洧녣20 = 210

      • 洧녣35 = (35 + 1) 洧논 35/2

        洧녣35 = 36 洧논 35/2

        洧녣35 = 630

        • Suku ke-12, suku ke-20, dan suku ke-35 pada pola bilangan segitiga berturut-turut adalah 78, 210, dan 630.
